COMP27112 — Image Processing, Lecture 7
"Blobs & Boundaries" — Interactive Visual Notes
================================================

HOW TO OPEN
-----------
Just open index.html in any modern web browser
(double-click it, or right-click > Open with > your browser).
Everything runs locally — no internet needed except for the
web fonts (it still works fine offline, just with fallback fonts).

WHAT'S INSIDE
-------------
index.html        the notes page
assets/app.js     all the interactive logic
assets/img/       all 56 lecture slides (slide-01 ... slide-56)

INTERACTIVE LABS
----------------
- Connectivity explorer (4- vs 8-connected)
- Connected Component Analysis step-through (slides 15–16)
- Animated boundary trace (slides 17–21)
- Chain-code direction compass
- Ordinary -> differential chain-code converter
- Perimeter calculator (E + √2·D)
- Chain-code -> area (Shoelace) calculator
- 12-question quiz (from the course quiz bank)
- 6 past-paper questions with model answers
- Full 56-slide deck gallery (Appendix)

Topics: blob finding, region labelling, boundary detection,
moments of area, chain codes, perimeter & area, blob tracking.
