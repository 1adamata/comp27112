COMP27112 — Chapter 8: File Formats, Exposure & Compression
Interactive visual notes (Lecture 8.1 & 8.2, T. Morley)

HOW TO OPEN
  Open  index.html  in any modern web browser (double-click it).
  Everything works offline — all images are in the assets/ folder
  and the threshold demo image is embedded in the page.

WHAT'S INSIDE
  index.html          the notes site (all CSS + JS inline)
  assets/img/*.png    slide photos & rendered slides used throughout

INTERACTIVE BITS
  • Pack 8 pixels into 1 byte (1-bit PBM)
  • Live thresholding of the tadpole image (drag T)
  • Adjustable camera aperture (iris) + depth-of-field crossfade
  • Shutter-speed crossfade (motion blur vs frozen)
  • GIF palette lookup + 3-3-2 packed-colour builder
  • Alpha-transparency slider
  • Run-length encoder (good vs bad case, live ratio)
  • JPEG pipeline step-through (DCT → quantise → zig-zag → decode)
  • JPEG quality slider with file sizes + blockiness toggle
  • 12-question self-test quiz

NOTES ON SOURCES
  • Built from the Lecture 8 slides, the lecture transcripts and the
    course quiz bank.
  • The IP handout's Chapter 6 (pp.110–122) covers REGION DESCRIPTION
    (chain codes, moments, compactness) — a different topic — and the
    handout does not cover file formats / exposure / compression at all,
    so those notes are based on the slides.
  • In the past papers, this material is examined only through the
    restricted multiple-choice section (Q1–Q20); the quiz here is built
    from the matching quiz-bank questions.
