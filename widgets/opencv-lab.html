Same function names, same argument order. The JS differences: you pass new cv.Size(k,k) instead of Size(k,k), constants live on cv.*, and you must .delete() every Mat (no garbage collection for WASM memory).

"Outline the structure of an OpenCV program." Answer: read (imread) → check it loaded → process (a function call or per-pixel loop over rows then columns) → write/show (imwrite). Note an image is stored in a Mat, indexed (row, col).
