COMP27112 — Chapter 5: The Graphics Pipeline — Interactive Visual Notes
=======================================================================

HOW TO OPEN
  Just double-click  index.html  (opens in any modern browser).
  Everything runs offline — no internet needed except the Google Fonts
  (it falls back to system fonts if you're offline).

WHAT'S INSIDE
  index.html      the full interactive notes
  assets/         30 slide images rendered from the lecture PDF

INTERACTIVE BITS
  • Scroll-reveal animations + reading-progress bar
  • Tap-to-expand graphics API cards (OpenGL/WebGL/three.js/Vulkan/DirectX/Metal)
  • Clickable vertex attribute explorer
  • Fixed vs Programmable pipeline toggle — click any stage to inspect it
  • Tabbed WebGL vs three.js triangle code
  • Tap-to-highlight GLSL qualifiers (attribute / uniform / varying)
  • LIVE WebGL shader playground — scrub uTime, move the mouse
  • Self-marking quiz (pipeline questions from the course bank)
  • Past-paper questions with reveal-on-click model answers

SOURCES
  Lecture "The Graphics Pipeline" (D. Petrescu) + transcript,
  the course Visual Quizzes, and COMP27112 past papers 2016–2025.
  Slide images © University of Manchester — included for personal study.
