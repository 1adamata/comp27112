COMP27112 — Viewing 1 (Chapter 4): Interactive Visual Notes
===========================================================

HOW TO OPEN
  Double-click  index.html  (opens in any modern browser).
  Keep the  assets/  folder next to index.html — it holds the slide images.
  Works fully offline. (Fonts load from Google Fonts when online; a clean
  fallback is used otherwise.)

WHAT'S INSIDE
  index.html        the notes site (all sections, CSS + JS embedded)
  assets/*.jpg      29 key visuals rendered from the lecture slides

INTERACTIVE BITS
  • Scroll-reveal sections + reading-progress bar
  • "Drag the eye" — move E around target C and watch f̂ = normalise(E − C)
     and the perpendicular ŝ update live
  • "Collapse the z axis" — animate orthographic projection flattening depth
  • "Which Pringles is rotated?" reveal (duality)
  • 7 self-marking quiz questions (from the unit quiz bank)
  • 6 past-paper questions with "Show approach" model answers

SOURCES
  Lecture slides + transcript (Viewing 1), the unit quiz bank, and
  COMP27112 past papers (2016, 2017, 2019, 2022).
