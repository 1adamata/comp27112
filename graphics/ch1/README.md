# COMP27112 — Chapter 1: Linear Algebra for Graphics
Interactive visual study notes.

## How to use
Open `index.html` in any modern browser (double-click it). Everything runs
locally — no install needed. An internet connection just makes the display
fonts (Fraunces / Spectral / JetBrains Mono) load; without it, the page still
works with system-font fallbacks.

## What's inside
- `index.html` — the full interactive notes page.
- `assets/` — every image used, pulled from the course presentation:
  - `suzanne.png`, `cheese.png` — Blender monkey + cheese (points vs vectors)
  - `slide_vectors_axes.png` — orthonormal axes + v = [2 1 0]ᵀ
  - `slide_dot_product_circle.png`, `trig_circle.png` — projection / trig circle
  - `slide_cross_product.png`, `crossproduct_hand.png` — right-hand rule
  - `slide_matrix_addition_thanos.png`, `thanos.png` — matrix addition sizing
  - `slide_linalg_homer.png`, `homer.png`, `slide_discussion_points.png`,
    `neo.png`, `manchester_logo.png` — section imagery

## Interactive bits
- Drag Suzanne to translate a point (P→P′) while v stays fixed
- Scalar-multiply slider
- Drag-the-vectors addition / parallelogram resultant
- Magnitude + unit-vector calculator
- Rotate-θ dot-product demo (a·b = cos θ, projection d)
- Right-/left-hand cross-product toggle
- "Matrix as a function" — edit A, drag v, watch Av
- 8 self-marking quiz questions (from the course visual quizzes)
- Expandable past-paper questions (2016–2024) with how-to-approach notes
