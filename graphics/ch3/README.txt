COMP27112 — Chapter 3: Transformations — Interactive Visual Notes
=================================================================

HOW TO OPEN
-----------
Double-click  index.html  to open it in any web browser.
Everything works offline. Keep the  assets/  folder next to index.html.

WHAT'S INSIDE
-------------
index.html        The full interactive notes page (scroll through it).
assets/           32 slide images from the lecture deck
                  (COMP27112_2Dtransformations), numbered in reading order.

FEATURES
--------
- Scroll-driven reveals + reading-progress bar + section nav.
- Click any slide image to enlarge it (Esc or click to close).
- Rotation demo: drag the angle slider, watch v -> v' and live cos/sin.
- Transformation Lab: drag scale / rotate / shear / translate sliders and
  toggle reflections; the live 3x3 homogeneous matrix updates as you go.
- "Order matters" demo: toggle Rotate->Translate vs Translate->Rotate.
- 8 tap-to-answer quiz questions (from the visual quizzes).
- 5 past-paper questions (2016-2025) with worked answers — click to expand.

SOURCES
-------
Lecture deck COMP27112_2Dtransformations (D. Petrescu, Univ. of Manchester),
the module video transcript, the visual quizzes, and past papers 2016-2025.
Built for personal revision.
