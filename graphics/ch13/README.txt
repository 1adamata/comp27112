COMP27112 — Chapter 13: Rendering Techniques
Interactive Visual Notes
=============================================

HOW TO OPEN
-----------
Unzip this folder, then double-click  index.html  to open it in any modern
web browser (Chrome, Firefox, Edge, Safari). No internet connection is
required except to load the web fonts (it still works offline, just with
fallback fonts).

WHAT'S INSIDE
-------------
index.html        — the page (structure + all styling)
app.js            — all content + the interactive components
assets/slides/    — 22 images exported from the RenderingTechniques lecture deck

SECTIONS
--------
00  The rendering pipeline      (tap each stage to expand)
01  Shadow mapping              (drag the light to test shadowing)
02a The reflection vector       (drag L and V; watch the specular meter)
02b Environment mapping         (equirectangular, light probes, cube maps)
03  Bump mapping                (drag the light; toggle Flat / Bumpy / Height)
04  Normal maps                 (tilt the normal, read the RGB it encodes)
05  Displacement mapping        (toggle Normal-map vs Displacement silhouette)
06  Quiz                        (questions from the course quiz bank)
07  Past exam questions         (2016–2025, with model-answer sketches)
08  Epilogue

SOURCES
-------
Lecture slides:  RenderingTechniques (D. Petrescu)
Quiz questions:  course visual-quiz bank (rendering-relevant subset)
