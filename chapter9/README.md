# The Hough Transform — Interactive Visual Notes
**COMP27112 · Introduction to Visual Computing · Image Processing (Lecture 9)**

Open `index.html` in any modern web browser (double-click it). No internet
connection is strictly required — only the web fonts load from the network; the
notes work fully offline with system-font fallbacks.

## What's inside
- A scroll-through study guide covering the whole Hough Lines chapter.
- **Lab 01** – interactive point⇄line duality canvas (click to add points).
- **Lab 02** – step through the Cartesian accumulator, point by point.
- **Lab 03** – spin the polar normal to see `r = x cos θ + y sin θ`.
- **Lab 04** – step through the polar accumulator.
- 5 self-marking quiz questions (from the module visual quizzes).
- A past-paper exam question (2022, Q32) with a model-answer skeleton.
