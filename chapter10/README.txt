COMP27112 — Binary Morphology (Image Processing, Lecture 10)
Interactive Visual Notes
============================================================

HOW TO OPEN
-----------
Double-click  index.html  to open it in any modern web browser
(Chrome, Firefox, Edge, Safari). No installation or server needed.
An internet connection lets the page load its display fonts; without
one it falls back to system fonts and still works fully.

WHAT'S INSIDE
-------------
index.html   The notes — one self-contained page.
assets/      All images used, taken from the course presentation:
               structuring_elements, the white-blood-cell
               original/threshold/erosion/dilation/opening/closing
               examples, and the opening-vs-closing worked grid.

INTERACTIVE BITS
----------------
- Drag the green handles on every before/after image.
- Morphology Playground: click or drag to draw pixels, choose an
  operation (Erode / Dilate / Open / Close) and a structuring
  element, then Play or Step the sweep and watch the output build.
- Quiz: tap an option for instant marking + feedback.
- Exam practice: attempt each question, then reveal the model answer.
